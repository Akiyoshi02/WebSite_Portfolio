This is the first post on the new portfolio. The site is built with **Astro**,
deployed as a fully static bundle, and ships effectively no JavaScript on
content pages by default.

Plans from here:

- **Project case studies** under `/projects/<slug>`, written in MDX so they
  can mix prose, code, and embedded components.
- **Engineering notes:** short, focused posts on building software with
  real users in mind: backend, AI integration, system design.
- **A `/uses` page** for the tools, hardware, and setup I work with.

If you spotted something you want to talk about, the contact link in the
nav goes straight to my inbox.
